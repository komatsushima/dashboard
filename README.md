# 📊 MCJP 印税明細ダッシュボード

MCJPから届く**印税支払明細書（著作権）PDF**をドラッグ＆ドロップするだけで、収益を自動分析・可視化するダッシュボードです。

---

## ✨ 機能

- **PDFをドラッグ＆ドロップ**するだけで即分析（アップロード不要・完全ブラウザ処理）
- **今期の受取金額**をひと目で確認
- **楽曲別ランキング**（どの曲が一番稼いでいるか）
- **プラットフォーム別内訳**（TikTok / Spotify / YouTube など）
- **カテゴリ別内訳**（ストリーミング / ダウンロード / 放送 / カラオケ）
- 専門用語の**用語集**付き（印税のしくみをわかりやすく解説）
- **Google サイトへの埋め込み**対応

---

## 🔒 プライバシーについて

PDFのデータは**外部サーバーに一切送信されません**。すべてブラウザ内で処理されます。

---

## 🚀 使い方

### 方法① そのままブラウザで使う（一番かんたん）

👉 **[ダッシュボードを開く](https://komatsushima.github.io/dashboard/dashboard.html)**

1. 上のリンクを開く
2. MCJPから届いたPDFをドラッグ＆ドロップ
3. 自動で分析される

---

### 方法② 自分のGitHub Pagesで公開する

1. このリポジトリの `dashboard.html` をダウンロード
2. 自分のGitHubリポジトリを作成して `dashboard.html` をアップロード
3. Settings → Pages → Branch: `main` → Save
4. 数分後に `https://あなたのユーザー名.github.io/リポジトリ名/dashboard.html` でアクセス可能

---

### 方法③ Google サイトに埋め込む

1. 方法②でURLを取得する
2. Google サイトの編集画面 → **埋め込み** → URLを貼り付け
3. 公開する

---

## 📄 対応PDFについて

MCJPが発行する以下の形式に対応しています：

- **印税支払明細書（著作権）** — 毎期届くPDF

> ⚠️ 他の会社・フォーマットのPDFには対応していません
